Introduction

This is a stand-alone quarto document with code chunks and documentation for an application preparing raw input data for MOPO EU/ Industrial cluster case study directly from AIDRES database.

Using the application

The files are produced using R and RStudio. Though the scripts can be executed using command line, it is recommended to use RStudio for rendering.

Clone this repository in your computer. Open the mopo_repo.Rproj (RStudio needs to be installed) file. Open refinery_analysis.qmd and click on render button (Crtl + Shift + K).

Loading libraries

The application checks whether the required packages for running the scripts are installed or not. If not, it automatically installs them and loads them. If they are already installed, it just loads them.

Get extra data

Get data on the CO2 consumption of co electrolysis processes (based on Figure 12 and 13 of the final aidres report) + data on the processes to convert hydrogen to Fischer-Tropsch liquid (FT) and hydrogen to methanol (MeOH) based on the processes ‘Hydrogen to Jet’ and ‘Methanol from hydrogen and carbon dioxide’, respectively, of the Danish technology catalogue.

For the AIDRES processes (COEL)FT-MEA and (COEL)MeOH-MEA, the CO2 consumption from the final aidres report is expressed in ton CO2/MWh LLF, which is converted to ton CO2/MWh LLF based on a lower heating value of 11.91 MWh/t. For ((COEL)MeOH)O-MEA, ton CO2/ton LLF is converted to ton CO2/ton olefins by dividing by 2.1552 ton MeOH/ton LLF and by dividing by 0.4377 ton olefins/ton MeOH. Check the ‘extra_data_processed.xlsx’ file for more info on the calculation.

For the ‘Hydrogen to Jet’ process in the Danish technology catalogue, the capex is expressed in Meur_(MW_FT)_year. To convert this to (eur-yr)/mwh, we multiply by 1000000 eur/Meur and divide by 8760 h/y. The fom is expressed in ‘eur_(MWh_FT)’ which is already the correct unit. Check the ‘extra_data_processed.xlsx’ file for more info on the calculation. –> Remark: we are still going to check these values with our TIMES model!

For the ‘Methanol from hydrogen and carbon dioxide’ in the Danish technology catalogue, the capex is expressed in Meur_(MW_methanol). To convert this to (eur-yr)/mwh, we multiply by 1000000 eur/Meur and divide by 8760 h/y. The fom is expressed in ‘keur_(MW_methanol_year)’, which is multiplied by 1000 eur/keur and divided by 8760 h/year to convert it to eur/mwh. Check the ‘extra_data_processed.xlsx’ file for more info on the calculation. –> Remark: we are still going to check these values with our TIMES model!
